# Rstudio-themes-inspired-by-Rebecca-das-musical
three r-studio themes (Orchideen, Nebelreich, and Wilde Freie See) created based on the stage design and lyrics of the [Vienna production](https://www.musicalvienna.at/en/schedule-and-tickets/schedule/production/1052/REBECCA/calendar) 
of **Rebecca das Musical**

[pinterest page](https://www.pinterest.jp/ideas/rebecca-musical/917491937791/) from which I found the images which the colour scheme used


Code based on `Monokai` in [tmTheme editor](https://tmtheme-editor.glitch.me/#!/editor/theme/Monokai)
